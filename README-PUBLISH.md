# Nova Command Website

Site statique pret a publier pour Nova Command.

## Fichiers a envoyer sur GitHub Pages

Quand tu modifies le site, envoie uniquement les fichiers qui ont change :

- `index.html`
- `styles.css`
- `demo.js`
- `robots.txt`
- `sitemap.xml`
- `assets/` seulement si une image change
- fichiers Google/Bing de validation, a garder a la racine

Les fichiers actuels de l'app sont dans la GitHub Release `v1.2.0`. Ajoute obligatoirement :

- `Nova-Command-Setup-1.2.0.exe`
- `Nova-Command-Setup-1.2.0.exe.blockmap`
- `latest.yml`

Ces trois fichiers permettent l'installation, la reinstallation et les mises a jour automatiques.
Le ZIP portable peut aussi etre joint, mais il ne se met pas a jour automatiquement.

## URL publiques

- Site : `https://strideur.github.io/`
- Sitemap : `https://strideur.github.io/sitemap.xml`

## Test local

Ouvre `index.html` dans le navigateur, ou lance un serveur statique depuis ce dossier.
